git clone或者push提示用户或者密码错误，但是不弹出输入框解决办法

发表于 2019-03-28 | 阅读次数:

今天遇到git push老是提示用户名密码错误，但只弹出一次输入用户名密码框，在我输错之后再也不弹出了，呜呜呜。

在一番百度之后，找到了可靠的解决办法，这里记录一下

首先：

进入控制面板，windows + R 然后输入control ， 如下图

点击“管理windows凭证”

可以看到这里将你的凭证设置为永久生效的，这就是为什么提示用户名密码错误却不弹出输入框的原因

解决办法：

将你仓库对应的凭证删除，之后再 git push时就会弹出输入框了。
