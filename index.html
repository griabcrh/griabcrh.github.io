解决百度云盘下载限速问题

发表于 2018-04-21 | 阅读次数:

相信有很多小朋友和我一样，被百度云盘下载限速的问题困扰的头疼，如果说下载的是1G之内的，倒还可以接受，一旦下载大文件，哎吆码！等死人有木有？

1.首先打开百度网盘 https://pan.baidu.com， 找到需要下载的资源，比如图片中的内容

2.按键盘F12，在弹出调试页面，然后选择Network–>All，点击Clear，如图片步骤

3.点击百度网盘资源后面的下载按钮，并且在调试页面中可以抓取的请求，并且点击download?sign那个链接

4.点击Response标签，在底部把页面往右拖，复制dlink后面一段http://….，复制双引号里面的全部内容

5.将复制的链接在浏览器中打开

会出现“无法访问此网站”的提示,在浏览器地址，把 https://d.pcs.baidu.com//file//840... 中后面两个斜杠都去掉一个，变成 https://d.pcs.baidu.com/file/840... ,点击回车，可以看到弹出直接下载页面和使用迅雷下载（猎豹浏览器，自带迅雷），点击迅雷下载或者点击直接下载，获取链接地址，然后去迅雷中新建下载。你也可以在迅雷里新建下载连接，将链接放进去下载。

至此，关于解决百度云盘下载限速问题已讲解完毕，网上有很多关于破解百度云盘下载限速教程，可自行百度。

一、前期准备

1.1 相关网站

在搭建个人博客的过程中，你可能会使用到下面几个网站。在这几个网站中都有相应的官方文档及教程。如果官方文档不能满足你，那么请Google。

二、搭建Git环境

2.1 注册Github

进入Github 网站，按照提示进行注册，然后登录。

登录完成之后，在你的主页点击图标 New Repository 创建一个新的版本库，因为我们是使用 Github Pages 去搭建我们的静态博客，所以版本库的名称应该是你的用户名+.github.io。如：我的用户名是：griabcrh，那么版本库的名字应该是： griabcrh.github.io ，这个是一定不能出错的。因为之后你将要访问的你的博客地址就是： https://griabcrh.github.io 。

至此，Github账号创建完成，GIthub Pages 所需要的版本库也创建好了。

进入 Git 官网，下载相应的 Git 版本，下载完成之后按照引导安装 Git 。安装完成之后在开始菜单中会有一个 Git Bash 。这是一个类似于Liunx的终端，在里面可以模拟Linux下的终端进行操作。
